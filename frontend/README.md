### Front End

A Next.js chat interface built with the App Router, Tailwind CSS, and TypeScript.

## Installation

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

## Running Locally

1. Make sure your FastAPI backend is running at `http://127.0.0.1:8000`

2. Start the Next.js development server:
```bash
npm run dev
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the production bundle
- `npm start` - Start the production server (after building)
- `npm run lint` - Run the linter

## Features

- Clean, minimal UI with a calm, outdoors-inspired design
- Real-time chat interface with the FastAPI backend
- Loading states and error handling
- Responsive design with Tailwind CSS
- TypeScript for type safety
